## Day 3
1. 数组
	- 一维数组
		- 声明：类型[ ]
		- 内存模型
			- 内存>JVM
				- 栈
					- main
					- 存储地址（引用）
				- 堆
					- new
					- 回收-占位
					- 地址索引-存储值
		- 操作
			- 下标：0开始
			- 长度：对象.length()
			- 遍历-循环
	- 二维数组
		- 声明：类型`[][]` （顺序随意），
			- 可以仅创建一维长度（一维的值为空）
			- 若要创建二维长度，必须创建一维
			- 二维长度可以再次new来自定义长度
		- 内存模型
			- 同上，一维存储值为数个一维数组的首地址，二维存储值才是内部值。
		- 操作
			- 下标：两个维度都从0开始
			- 长度：两个维度的长度独立
			- 遍历-二重循环
## Day 4
1. 面向对象
	- 类
		- 类的成员
			- 属性：成员变量
			- 行为：方法实现
		- 存储结构
			- 方法区：
				- 类似堆
				- 只进（类及其父类第一次加载）不出
				- 子类对象->父类对象的方法依次压栈。
	- 对象：可以调用类所有的成员
		- 构造器（不是方法）：初始化
		- this
	- 高级特性：
		- 封装
			- 利用访问权限控制符
		- 继承
			- 子类内部可以调用super对象（和super()，注意要在最顶端），但子类对象不能调用super对象和（super()）
		- 多态
			- 访问权限控制符
				- public
				- protected
				- default
				- private
