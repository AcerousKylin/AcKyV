## Day 3
1. 数组
	- 一维数组
		- 声明：类型[ ]
		- 内存模型
			- 内存>JVM
				- 栈
					- main
					- 存储地址（引用）
				- 堆
					- new
					- 回收-占位
					- 地址索引-存储值
		- 操作
			- 下标：0开始
			- 长度：对象.length()
			- 遍历-循环
	- 二维数组
		- 声明：类型`[][]` （顺序随意），
			- 可以仅创建一维长度（一维的值为空）
			- 若要创建二维长度，必须创建一维
			- 二维长度可以再次new来自定义长度
		- 内存模型
			- 同上，一维存储值为数个一维数组的首地址，二维存储值才是内部值。
		- 操作
			- 下标：两个维度都从0开始
			- 长度：两个维度的长度独立
			- 遍历-二重循环
